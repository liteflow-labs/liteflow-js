# Chat

The Chat template offers a ready-to-use chat application, demonstrating the full capabilities of Liteflow’s wallet-to-wallet messaging. It's designed for easy integration and customization within your existing infrastructure.

## Features included

<Cards>
  <Card title="Direct Messaging">
    Wallet-to-wallet communication baked into your application.
  </Card>
  <Card title="Theme Customization">
    Tailor the chat interface to complement your brand’s aesthetics.
  </Card>
  <Card title="Versatile Widget">
    An embeddable chat widget that can be placed in any part of your
    application.
  </Card>
  <Card title="Multi-Chain Compatibility">
    Compatible with major chains including Ethereum, Polygon, Binance Smart
    Chain, and LightLink.
  </Card>
  <Card title="Wallet Compatibility">
    Support for unlimited wallets thanks to
    [RainbowKit](https://www.rainbowkit.com/)
  </Card>
  <Card title="Open Source">
    Open-source and fork-able. You can view the source code and contribute to it
    on GitHub at [liteflow-labs/chat](https://github.com/liteflow-labs/chat).
  </Card>
</Cards>

## Libraries used

The Chat template is based on the following tech stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Liteflow](https://liteflow.com/)
- [ChakraUI](https://chakra-ui.com/)
