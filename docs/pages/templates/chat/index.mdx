The chat template is a fully fledged chat application showcase the uses of the wallet-to-wallet messaging feature of the Liteflow infrastructure.

## Libraries used

The Chat template is based on the following tech stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Liteflow](https://liteflow.com/)
- [ChakraUI](https://chakra-ui.com/)
