# Marketplace

The Marketplace template is a fully fledged NFT marketplace application built on Liteflow infrastructure. This template provides an operational example of our system and serves as a starter kit for rapid product development.

Explore a live demonstration of the marketplace template at: https://demo.liteflow.com.

## Features included

<Cards>
  <Card title="Marketplace Core">
    NFT Explorer, NFT Minter, and Home Page Customization.
  </Card>
  <Card title="Discovery Tools">Search System and Trait Filters for NFTs.</Card>
  <Card title="Transaction Types">
    Fixed Price, Open Offers, Native Token, and ERC20 Support.
  </Card>
  <Card title="ERC1155 Support">Partial Fill Functionality.</Card>
  <Card title="User Experience">
    Profiles, Wallets, Dashboards, Verification, and Reporting System.
  </Card>
  <Card title="Engagement">
    In-app and Email Notifications via Webhook, Wallet-to-Wallet Messaging.
  </Card>
  <Card title="Compatibility">
    Support for Multiple Wallets, Chains, Currencies, and Languages.
  </Card>
</Cards>

## Libraries used

The Marketplace template is based on the following tech stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Liteflow](https://liteflow.com/)
- [ChakraUI](https://chakra-ui.com/)
- [Wagmi](https://wagmi.sh/)
- [Rainbowkit](https://rainbowkit.com/)
- [Next translate](https://github.com/aralroca/next-translate)
