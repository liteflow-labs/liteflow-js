# Wallet-to-Wallet Messaging

Wallet to Wallet Messaging solution, leverages the [XMTP protocol](https://xmtp.org/) to facilitate direct and secure communication across wallets. This feature is versatile, supporting social, commercial, and support interactions, and can be incorporated into any application or used via our dedicated chat template.

## How to Use It

<Steps>

### Marketplace Integration

The messaging system comes pre-integrated with our [marketplace template](/template/marketplace), streamlining setup.

### Standalone Application Setup

Deploy messaging as a separate app using our [chat template](/template/chat) for specialized communication platforms.

### Custom Application Integration

Utilize the [`@nft/chat` library](/tools/chat) to embed the [`Chat` component](/tools/chat) directly into your application for a personalized implementation.
