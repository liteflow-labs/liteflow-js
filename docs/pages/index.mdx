# Liteflow

Liteflow is a comprehensive infrastructure designed specifically for applications based on Non-Fungible Tokens (NFTs). It provides a dedicated API and customizable infrastructure to facilitate the creation and trading of NFTs.

Liteflow offers a suite of tools designed to help businesses launch their NFT Marketplaces swiftly and efficiently.

## Tools

To interact with these products, Liteflow provides a set of tools from UI to monitor all the activity to low level developer tools to ensure everything can be fully customized for your business needs.

<Cards num="3">
  <Card href="/tools/dashboard" title="Dashboard" icon={<ChartBarIcon />} arrow>
    The dashboard is the main interface for managing your NFT marketplace. It
    allows you to create and manage collections, monitor activity, and configure
    your marketplace.
  </Card>
  <Card href="/tools/graphql-api" title="GraphQL API" icon={<LinkIcon />} arrow>
    This is the main interface for interacting with the Liteflow infrastructure.
    It allows you to query and manipulate data in your NFT marketplace.
  </Card>
  <Card href="/tools/webhook" title="Webhook" icon={<CommandLineIcon />} arrow>
    Webhooks provide a way for your application to be notified of events
    happening in your NFT marketplace. This is useful for triggering actions
    based on specific events.
  </Card>
  <Card href="/tools/service" title="Service" icon={<CloudIcon />} arrow>
    Services are the building blocks of your NFT marketplace. They encapsulate
    business logic and can be used to tailor your product based on your business
    model.
  </Card>
  <Card
    href="/tools/sdk-core"
    title="SDK Javascript"
    icon={<CodeBracketIcon />}
    arrow
  >
    This SDK provides a JavaScript interface for interacting with the Liteflow
    infrastructure. It can be used in any JavaScript-based application.
  </Card>
  <Card
    href="/tools/sdk-react"
    title="SDK React"
    icon={<CodeBracketSquareIcon />}
    arrow
  >
    This SDK is specifically designed for React applications. It provides React
    components and hooks for interacting with the Liteflow infrastructure.
  </Card>
</Cards>
